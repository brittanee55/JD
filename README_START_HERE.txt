Human Knowledge Atlas - Complete Static Build

How to run:
1. Unzip this folder.
2. Open the folder in VS Code.
3. Right-click index.html and choose Open with Live Server.
4. Do NOT open by double-clicking the HTML file, because JSON fetches need Live Server.

Included:
- Complete public pages
- Working 3D globe with hover-stop rotation
- data/*.json Atlas database files
- Transcript Library and transcript viewer
- User-provided transcript files in /transcripts
- Assets/images/logos/icons folders with generated web graphics
- Demo authentication pages using localStorage
- Demo admin dashboard using localStorage
- API helper files for future Firebase/AI integration

Important: Transcripts are source-review material. Verify claims before treating them as facts.
