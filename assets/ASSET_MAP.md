# Human Knowledge Atlas Asset Map

This finished build includes custom generated visual assets for the full site/app: heroes, wing covers, globe textures, episode cards, people/profile placeholders, books, concepts, timeline art, courses, journal art, blog/newsletter art, logo and favicon.

The art direction is designed for Julian-style long-form podcast research: dark museum archive, celestial maps, antique gold, emerald glow, source documents, knowledge networks, and a cinematic globe.

These are original generated placeholder assets and can be replaced later with licensed guest photos, official episode thumbnails, or production photography.
