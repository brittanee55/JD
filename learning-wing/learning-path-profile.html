Overview

What You Will Learn

This learning path helps users understand how intelligence institutions developed, how they connect to Cold War history, why secrecy and oversight matter, and how to evaluate podcast claims using source literacy.

Skill

Source literacy and historical thinking.

Focus

Intelligence agencies, records, and public accountability.

Outcome

Create a safer, source-aware episode summary.

Final Challenge

Create a Source-Aware Episode Summary

Final Project

Choose one episode about intelligence. Write a summary that clearly separates documented facts, guest claims, disputed interpretations, and open questions.

  1. Summarize the episode in your own words.
  2. Identify three claims that need source review.
  3. Find one book or official source for context.
  4. Write what remains uncertain.
  5. Create three journal prompts for future reflection.
